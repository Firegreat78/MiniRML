Инструментальная система разработки программ управления промышленными роботами. Разработана с использованием Qt 6.8.2

В данном проекте:
- разработан интерпретируемый C-подобный язык программирования для управления простым роботом-манипулятором;
- разработан графический интерфейс, показывающий отладочную информацию (текущие значения переменных в данной области видимости и типы этих переменных; координаты шарниров в пространстве; выполненные шаги выполнения).

Робот-манипулятор, управляемый языком программирования, представляет собой совокупность шарниров, соединённых прямыми сегментами. Исходное состояния робота-манипулятора с 6 сегментами представлено на изображении ниже.

Язык программирования имеет следующие типы данных:
- `short` - знаковое целое число, занимающее 2 байта. Число, представляемое этим типом данных, может принимать значение в диапазоне [-32768; 32767];
- `int` - знаковое целое число, занимающее 4 байта. Число, представляемое этим типом данных, может принимать значение в диапазоне [-2,147,483,648; 2,147,483,647];
- `long` - знаковое целое число, занимающее 8 байт. Число, представляемое этим типом данных, может принимать значение в диапазоне [-9,223,372,036,854,775,808; 9,223,372,036,854,775,807];
- `double` - число с плавающей запятой двойной точности по IEEE 754;
- `bool` - булевый тип.

Реализовано приведение типов между целочисленными типами данных (`short`, `int`, `long`).

Control flow реализован с использованием операторов `if` и `while`.

Для управления роботом-манипулятором используются команды `Move` и `Rotate`.

Команда `Move`(`x`, `y`) поступательно сдвигает все шарниры робота. Аргументы, передаваемые в команду, должны иметь тип `double`.

Описание аргументов:
- `x` - смещение по оси X;
- `y` - смещение по оси Y.

Команда `Rotate`(`i`, `nX`, `nY`, `nZ`, `φ`) поворачивает робота за шарнир на заданный угол в радианах вокруг некоторой оси. 

Описание аргументов:
- `i` - индекс шарнира, за который осуществляется поворот (тип аргумента: short);
- `nX` - компонента X направляющего вектора оси, вокруг которой осуществляется поворот других шарниров (тип аргумента: double);
- `nY` - компонента Y направляющего вектора оси, вокруг которой осуществляется поворот других шарниров (тип аргумента: double);
- `nZ` - компонента Z направляющего вектора оси, вокруг которой осуществляется поворот других шарниров (тип аргумента: double);
- `φ` - угол поворота в радианах (тип аргумента: double).

### Графическая иллюстрация поворота

Для подсчёта новых координат используется формула поворота Родрига в трёхмерном пространстве. 

Пусть точка до поворота имеет координаты (`x`, `y`, `z`), а после поворота имеет координаты (`x'`, `y'`, `z'`). Вращение происходит вокруг оси с направляющим вектором (`u_x`, `u_y`, `u_z`), проходящей через точку (`p_x`, `p_y`, `p_z`). 

Тогда вычисление новых координат `(x', y', z')` производится по следующим уравнениям:

```math
\begin{aligned}
x' &= p_x + x \cdot \cos(\theta) + (u_y Z - u_z Y) \cdot \sin(\theta) + u_x (u_x X + u_y Y + u_z Z)(1 - \cos(\theta)) \\
y' &= p_y + y \cdot \cos(\theta) + (u_z X - u_x Z) \cdot \sin(\theta) + u_y (u_x X + u_y Y + u_z Z)(1 - \cos(\theta)) \\
z' &= p_z + z \cdot \cos(\theta) + (u_x Y - u_y X) \cdot \sin(\theta) + u_z (u_x X + u_y Y + u_z Z)(1 - \cos(\theta))
\end{aligned}

```

Где

```math
$$
X = x - p_x, \quad Y = y - p_y, \quad Z = z - p_z
$$
```
